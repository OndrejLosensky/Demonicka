# Dashboard

Průvodce používáním hlavního dashboardu aplikace Démonická.

## Přehled dashboardu

### Hlavní komponenty
- **Aktuální událost** - Název a základní informace o probíhající události
- **Statistiky** - Aktuální stav konzumace a statistiky
- **Seznam účastníků** - Přehled aktivních účastníků
- **Stav sudů** - Vizualizace stavu jednotlivých sudů

### Statistiky v reálném čase
- Celková konzumace
- Průměrná konzumace na osobu
- Počet aktivních účastníků
- Rychlost konzumace

## Práce s dashboardem

### Sledování události
- Průběžné statistiky se automaticky aktualizují
- Barevné indikátory stavu sudů
- Notifikace o důležitých událostech
- Časová osa aktivity

### Správa účastníků
- Rychlé přidání účastníka
- Označení aktivních/neaktivních účastníků
- Přehled individuálních statistik
- Rychlé akce pro správu

### Správa sudů
- Přidání nového sudu
- Sledování stavu naplnění
- Oznámení o docházejícím sudu
- Historie výměn

## Funkce pro organizátory

### Ovládací panel
- Zahájení/ukončení události
- Pozastavení sledování
- Resetování statistik
- Správa přístupů

### Monitoring
- Graf konzumace v čase
- Teplotní mapa aktivity
- Statistiky po hodinách
- Predikce spotřeby

### Nastavení
- Pravidla bodování
- Limity a upozornění
- Zobrazení statistik
- Notifikace

## Tipy pro efektivní použití

### Organizace události
- Pravidelně kontrolujte statistiky
- Sledujte trendy konzumace
- Reagujte na upozornění
- Udržujte aktuální data

### Optimalizace
- Používejte filtry pro lepší přehled
- Nastavte si důležité notifikace
- Využívejte rychlé akce
- Pravidelně aktualizujte stav

### Řešení problémů
- Kontrola připojení
- Aktualizace dat
- Řešení nesrovnalostí
- Kontaktování podpory
