# Vytvoření události

Události jsou základním stavebním kamenem aplikace Démonická. Každá událost představuje samostatné setkání s vlastními účastníky, sudy a statistikami.

## Jak vytvořit novou událost

### 1. Přístup k vytvoření události
- Přihlaste se do aplikace jako administrátor
- V hlavním menu klikněte na **"Události"**
- Klikněte na tlačítko **"Nová událost"**

### 2. Vyplnění základních informací
Při vytváření události budete potřebovat:

**Povinné údaje:**
- **Název události** - Např. "Pivní večírek 2024"
- **Datum a čas** - Kdy se událost koná
- **Místo konání** - Kde se událost koná

**Volitelné údaje:**
- **Popis** - Detailnější informace o události
- **Maximální počet účastníků** - Limit účastníků
- **Typ události** - Kategorie události

### 3. Konfigurace události
Po vytvoření můžete nastavit:

**Základní nastavení:**
- Aktivace/deaktivace události
- Změna názvu a popisu
- Úprava data a místa

**Pokročilé nastavení:**
- Automatické ukončení události
- Notifikace a upozornění
- Export dat

## Stav události

Událost může být v různých stavech:

- **Neaktivní** - Událost je vytvořena, ale není aktivní
- **Aktivní** - Událost běží a účastníci mohou konzumovat
- **Ukončená** - Událost je ukončena, data jsou uzamčena

## Co dál?

Po vytvoření události pokračujte na:
1. [Přidání uživatelů](add-users.md) - Zaregistrujte účastníky
2. [Správa sudů](../user-guide/barrels.md) - Přidejte sudy
3. [Dashboard](../user-guide/dashboard.md) - Sledujte průběh události

## Tipy

- **Pojmenování** - Používejte jasné a popisné názvy událostí
- **Datum** - Nastavte správné datum, později ho nelze změnit
- **Popis** - Přidejte užitečné informace pro účastníky
- **Testování** - Než událost aktivujete, otestujte nastavení
