# Základní funkce

## Přehled

Démonická aplikace poskytuje komplexní řešení pro správu událostí a sledování konzumace piva. Zde jsou hlavní funkce systému rozdělené do klíčových oblastí.

## Správa událostí

### Aktivní událost
- Vždy pouze jedna aktivní událost
- Automatické přiřazování účastníků a sudů k aktivní události
- Přepínání mezi událostmi
- Historie událostí

### Statistiky události
- Přehled konzumace v reálném čase
- Žebříček účastníků
- Analýza spotřeby
- Export dat

## Správa účastníků

### Registrace a profily
- Rychlá registrace účastníků
- Profilové informace
- Historie konzumace
- Osobní statistiky

### Kategorizace
- Rozdělení podle pohlaví
- Vlastní skupiny
- Týmové statistiky

## Správa sudů

### Evidence sudů
- Sledování stavu sudů
- Historie čepování
- Automatické upozornění při docházení

### Statistiky sudů
- Průměrná spotřeba
- Rychlost konzumace
- Predikce spotřeby

## Analytické nástroje

### Dashboard
- Přehledové statistiky
- Grafy a vizualizace
- Filtry a časové období
- Exporty reportů

### Žebříčky
- Celkové žebříčky
- Žebříčky podle kategorií
- Historické záznamy
- Týmové žebříčky

## Uživatelské rozhraní

### Responzivní design
- Optimalizace pro mobilní zařízení
- Přizpůsobení různým velikostem obrazovky
- Intuitivní ovládání

### Přizpůsobení
- Světlý/tmavý režim
- Jazykové mutace
- Uživatelské preference
